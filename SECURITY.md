# Security Policy

This repository (`AethelredFoundation/aethelred-sdk-ts`) follows the Aethelred Foundation
security program.

- Canonical public source: `AethelredFoundation/aethelred`
- Repo role: `sdk-typescript`
- Monorepo source path: `sdk/typescript/`

## Reporting A Vulnerability

Do not open public issues for vulnerabilities.

Email [security@aethelred.io](mailto:security@aethelred.io) with:

- affected repository and version or commit
- impact summary
- reproduction steps or proof-of-concept
- suggested mitigations if known
